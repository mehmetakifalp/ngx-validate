Stackblitz Example

Installation

bash yarn add @ngx-validate/core or bash npm install @ngx-validate/core

Usage

Import core module to your main module as follows:

```typescript import { BrowserModule } from '@angular/platform-browser'; import { NgModule } from '@angular/core'; import { ReactiveFormsModule } from '@angular/forms';

import { AppRoutingModule } from './app-routing.module'; import { AppComponent } from './app.component'; import { NgxValidateCoreModule } from '../../packages/core/src/public_api';

@NgModule({ declarations: [AppComponent], imports: [BrowserModule, AppRoutingModule, ReactiveFormsModule, NgxValidateCoreModule.forRoot()], providers: [], bootstrap: [AppComponent], }) export class AppModule {}

```
